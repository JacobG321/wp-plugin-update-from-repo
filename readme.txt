== Description ==

This is only for testing/learning.

== Frequently Asked Questions ==

= How does this plugin update from GitHub? =

The plugin uses the GitHub API to check for updates in the specified repository and updates the plugin if a new version is found.

= Can I use this plugin with private repositories? =

No, this version only supports public repositories. For private repositories, additional authentication is required.
